After extracting the file, copy niyo sha then lagay sa C:\xampp\htdocs\ then paste niyo sa loob lang ng htdocs

rename niyo yung file to "sbm"

Download Antigravity (mas prefer kase s'ya na mismo mag iinstall lahat ng need mo in just one prompt)

command lang yung antigravity na i-set up lahat ng dependencies and libraries needed then i-try niya if gagana yung ML and AI

after mag complete si antigravity sa lahat, pumunta sa folder ng sbm then open yung folder ng ML then hanapin yung "start_sbm_ml" then start as administrator, after mag open ng file na yan click enter lang para mag run yung ML.

then run niyo yung XAMPP, tapos punta ng browser then pumunta sa phpmyadmin then gawa kayo ng db and name it ng "sbm_db"

after gumawa db navigate lang yung import then "file to import"

hanapin niyo 'to "C:\xampp\htdocs\config\sbm_db"

after mag import paste niyo 'to sa ibang tab "http://localhost/sbm/index.php"
